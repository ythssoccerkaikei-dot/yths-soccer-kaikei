# スポーツクラブ会計管理システム

このアプリケーションは、スポーツクラブの会計管理を効率化するためのWebアプリです。

## 機能

- ✅ 年度管理
- ✅ ユーザー管理（会計担当・コーチ・トレーナー）
- ✅ 会員管理
- ✅ 月次会費管理（チェックボックス式）
- ✅ スタッフ在籍管理
- ✅ 会場管理
- ✅ 活動記録（コーチ・トレーナー用）
- ✅ 報酬管理（コーチ代・交通費・ETC代の自動計算）
- ✅ 収入・支出管理
- ✅ 年度収支決算レポート
- ✅ パスワード変更機能

## デプロイ手順

### 1. GitHubにアップロード

1. GitHubで新しいリポジトリを作成
   - リポジトリ名: `sports-club-accounting`（任意）
   - Public または Private を選択
   - README は追加しない（既に含まれています）

2. このプロジェクトフォルダ全体をアップロード
   - GitHub上で「uploading an existing file」をクリック
   - すべてのファイルをドラッグ&ドロップ
   - Commit message: "Initial commit"
   - 「Commit changes」をクリック

### 2. Vercelでデプロイ

1. [Vercel](https://vercel.com/) にアクセスしてログイン
   - GitHubアカウントでログイン推奨

2. 「New Project」をクリック

3. GitHubリポジトリをインポート
   - 先ほど作成した `sports-club-accounting` リポジトリを選択
   - 「Import」をクリック

4. プロジェクト設定
   - Framework Preset: **Next.js** （自動検出されます）
   - Root Directory: `.` （デフォルトのまま）
   - Build and Output Settings: （デフォルトのまま）
   - Environment Variables: （設定不要）

5. 「Deploy」をクリック

6. 数分待つと... **完成！** 🎉
   - デプロイ完了後、URLが発行されます
   - 例: `https://sports-club-accounting-xxxxx.vercel.app`

## セキュリティに関する注意

1. デフォルトパスワードは必ず変更する
2. 各ユーザーに固有のパスワードを設定する
3. パスワードは定期的に変更する
4. 本番環境では必ずHTTPS（Vercelは自動対応）

## データの保存について

このアプリはブラウザの localStorage を使用してデータを保存します。
- データはユーザーのブラウザに保存されます
- サーバーにはデータは送信されません
- ブラウザのキャッシュをクリアするとデータが消えます
- 定期的にデータのバックアップを推奨します
